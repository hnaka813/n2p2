virtual void setJacobian(doubleconst *const jacobian, std::size_tconst columns = 1) = 0¶

Set pointer to current Jacobi matrix.

Note

If there are \(\) errors and \(\) weights, the Jacobi matrix is a \(\) matrix stored in column-major order.

Parameters

  • [in] jacobian: Derivatives of error with respect to weights.

  • [in] columns: Number of gradients provided.
